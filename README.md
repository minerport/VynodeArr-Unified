# VynodeNew

VynodeNew is a clean-start unified Movies and TV media-management application.
Milestone N1 provides neutral domain contracts, isolated deterministic adapters,
an API gateway, and an original branded UI proof.

Requirements: Node.js 20+. Run `npm test`, `npm run verify`, or `npm start`, then
open `http://localhost:4310`.

The fixtures are intentionally read-only. They prove boundaries without requiring
production credentials or exposing source-engine APIs to the browser.
