# VynodeArr 0.4.9

VynodeArr 0.4.9 delivers a unified visual identity for the Movies and Television engines while preserving their existing workflows, APIs, settings, and isolated data.

## Highlights

- modernized native Movies and Television layouts with shared VynodeArr branding;
- distinct blue Movies and green Television accents with improved navigation, spacing, cards, and search presentation;
- restored native artwork and proxy-safe image loading;
- VynodeArr browser titles and favicon throughout both engine interfaces;
- dashboard Attention Center now lists detailed health messages from both native engines;
- responsive rendering remains compatible with cached dashboard responses during upgrades.

## Compatibility

- Windows x64, Linux x64, Docker x86-64, and Unraid x86-64 remain supported;
- both native engines remain pinned and packaged as a tested pair;
- movie and television databases, settings, queues, APIs, and media operations remain isolated;
- native engine self-updates remain unsupported; update VynodeArr as a complete package.
