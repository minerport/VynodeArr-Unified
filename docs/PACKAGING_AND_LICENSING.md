# Packaging and licensing

N2 packages only original VynodeNew code and connects over HTTP to separately
installed engines. No reviewed engine source, binary, container layer, artwork,
database, or frontend asset is copied or redistributed. This is the preferred
review mode and minimizes—but does not automatically eliminate—license risk.

Bundling a modified engine or distributing an image containing one requires
legal review of GPLv3 corresponding source, build/install information,
copyright/license notices, modification marking, written offers where
applicable, source/image version alignment, update distribution, and all
transitive licenses. Private repository visibility does not waive distribution
obligations when binaries are conveyed.

The Compose profile contains inert placeholders only. It proves future network
and lifecycle seams but includes no engine. Do not publish bundled images until
notices, source delivery, upgrade ownership, and trademark review are complete.
