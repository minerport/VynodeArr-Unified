# First run and accounts

An empty `/data` directory launches the setup wizard instead of login. The
administrator supplies display name, unique username, unique email, strong
password, and confirmation. Successful creation starts a remembered session
and marks setup complete by persisting the user. The wizard appears again only
if no users exist.

Existing installations accept username or email, password, and Remember Me.
Sessions store browser, operating system, masked IP, creation, last activity,
and expiry. Users can revoke the current session or all other sessions.

My Account supports display name, username, email, password, profile image URL,
time zone, date/time format, theme, and a language-ready preference. Password
changes require the current password and revoke every other session.

Administrators can create, enable/disable, role, reset, force-logout, and delete
users. The current account and last enabled administrator are protected.
